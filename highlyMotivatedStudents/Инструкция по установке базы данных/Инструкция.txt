Для работы с приложением требуется установленная Microsoft SQL Server (EXPRESS версия ядра)
Для полноценной работы с приложением требуется восстановить базу данных из данной папки
1. В меню "Обозреватель обьектов" нужно нажать ПКМ по "Базы данных" и выбрать из выпадающего меню "Востановить базу данных"
2. Далее следует выбрать устройство и затем нажать на кнопку "..." которая располагается правее
3. В появившемся меню следует нажать кнопку "Добавить" и указать расположение бд в данной папке
4. Далее нажать кнопку "Ок" и снова "Ок"
5. Если все действия были выполнены правильно то база данных будет восстановлена.
